*****Host Experience*****
1. Account Registration & Login
Hosts begin by creating an account through the "Register" page.
Once registered, hosts log in with username and password then they are directed to the Host Dashboard.
2. Host Dashboard
The Host Dashboard provides access to:
-Create Event
-View Events
-Log Out


3. Event Creation
When creating an event, hosts may enter:
Event title
Event date
Event time
Event image (optional)
Event description
Registry link (optional)
Upon submission, the event becomes available for guest invitations
on the event detail page. 

4. Event Management
Hosts can view all events they have created.
Selecting an event opens the Event Detail Page, where the host may:
Edit event details
Delete the event
Manage guest invitations
View current invitation status'

5. Guest Invitations
From the Manage Guests page:
Hosts invite guests by entering:
First name
Last name
Email address
The system checks for duplicate email addresses and alerts the host if the guest email already exists.
Each invited guest receives an email from YAY or NAY RSVP containing:
A unique RSVP code
A button with a direct link to that specific event page

6. Guest Tracking
A guest list table is generated below the invitation form and updates automatically as guests RSVP. 
The table displays:
Attendance status (Yes / No)
Meal selection
Plus-one status
Plus-one meal selection (if applicable)
Hosts may remove a guest and re-invite them if necessary.

*****Guest Experience*****
1. Receiving the Invitation
Guests receive an email invitation that includes:
Event details
A personalized RSVP code
A button linking directly to the event page

2. Viewing the Event
Guests click the link in the email to access the event page.
The page displays event information and a View Registry button 
if a registry link was provided.

3. RSVP Authentication
Guests enter their unique RSVP code on the event page.
Once authenticated, the RSVP form becomes available.

4. Submitting an RSVP
Guests may:
Select Yay or Nay
Choose a meal option
Indicate whether they are bringing a plus one
If selected, an additional meal option appears
Upon submission, the RSVP is saved and reflected in

5. Updating an RSVP
Guests can update their RSVP at any time by revisiting the 
event page and re-entering their RSVP code. Each new 
submission replaces the previous response.
